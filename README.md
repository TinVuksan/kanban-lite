## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/TinVuksan/kanban-lite.git
cd your-repo-name
npm install
```
### Running the project locally

```bash
npm run dev
```
